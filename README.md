# Fenan Dental Clinic Website

Modern, responsive website for Fenan Dental Clinic in Addis Ababa, built with React, TypeScript, and TanStack Start.

## Features

- 🦷 Modern dental clinic website
- 📱 Fully responsive design
- ⚡ Fast performance with TanStack Start
- 🎨 Beautiful UI with Tailwind CSS
- 🎯 SEO optimized with meta tags
- 📍 Location-based services information
- 📞 Easy contact and booking options

## Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/start)
- **UI Components**: Radix UI
- **Styling**: Tailwind CSS
- **3D Graphics**: Three.js with React Three Fiber
- **Animations**: Framer Motion
- **Routing**: TanStack Router
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js 18+
- Bun package manager (recommended)

### Installation

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Development

### Available Scripts

- `bun run dev` - Start development server (http://localhost:5173)
- `bun run build` - Build for production
- `bun run build:dev` - Build in development mode
- `bun run preview` - Preview production build
- `bun run lint` - Run ESLint
- `bun run format` - Format code with Prettier

## Project Structure

```
src/
├── routes/          # Page routes
├── components/      # Reusable UI components
├── hooks/          # Custom React hooks
├── lib/            # Utilities and helpers
├── assets/         # Images and static assets
└── styles.css      # Global styles
```

## Deployment

The website is optimized for deployment on Neftily and other modern hosting platforms.

### Build Output

- Production builds are generated in the `dist/` directory
- Fully static and can be served from any CDN
- Optimized for performance and SEO

### Environment Variables

No environment variables are required for the base deployment. The site works with default configuration.
